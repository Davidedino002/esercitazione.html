Attributi HTML Tutti gli elementi HTML possono avere attributi Gli attributi forniscono informazioni aggiuntive sugli elementi Gli attributi sono sempre specificati nel tag di inizio Gli attributi solitamente si presentano in coppie nome/valore come: nome="valore" L'attributo href Il tag definisce un collegamento ipertestuale. L' hrefattributo specifica l'URL della pagina a cui porta il collegamento: Esempio Visit W3Schools L'attributo src Il tag è utilizzato per incorporare un'immagine in una pagina HTML. L'src attributo specifica il percorso all'immagine da visualizzare:agine da visualizzare: Esempio Esistono due modi per specificare l'URL nell'attributo src : 1. URL assoluto : link a un'immagine esterna ospitata su un altro sito web. Esempio: src="https://www.w3schools.com/images/img_girl.jpg" . Note: Le immagini esterne potrebbero essere protette da copyright. Se non ottieni il permesso di utilizzarle, potresti violare le leggi sul copyright. Inoltre, non puoi controllare le immagini esterne; possono essere rimosse o modificate all'improvviso. 2. URL relativo - Link a un'immagine ospitata nel sito web. Qui, l'URL non include il nome di dominio. Se l'URL inizia senza una barra, sarà relativo alla pagina corrente. Esempio: src="img_girl.jpg". Se l'URL inizia con una barra, sarà relativo al dominio. Esempio: src="/images/img_girl.jpg". Suggerimento: è quasi sempre meglio usare URL relativi. Non si romperanno se cambi dominio. Usa sempre attributi in minuscolo Lo standard HTML non richiede nomi di attributi in minuscolo. L’attributo title (e tutti gli altri attributi) possono essere scritti con lettere maiuscole o minuscole come title o TITLE. Tuttavia, raccomandiamo nomi di attributi in minuscolo in HTML e richiede attributi in minuscolo per tipi di documento più rigorosi come XHTML. Usa sempre valori degli attributi tra virgolette Lo standard HTML non richiede virgolette intorno ai valori degli attributi. Tuttavia, raccomandiamo le virgolette in HTML e richiede virgolette per tipi di documento più rigorosi come XHTML.
