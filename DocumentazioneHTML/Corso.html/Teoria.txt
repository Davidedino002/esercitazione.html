cos'è la programmazione 
La programmazione consiste nel dare istruzioni ad una macchina 
Le istruzioni possono essere procedurali o ad oggetti
Le istruzioni possono essere di tipo procedurale o ad oggetti
Procedurali                                          AD oggetti
                                 
istruzioni in blocchi eseguite in sequenza          Istruzioni in oggetti che interagiscono tra loro 
                                                    indipendentemente dall'ordine 

                                cos'è la programmazione  
La programmazione puo' essere a basso livello o ad alto livello
Basso Livello indica un codice con una sintassi piu' vicina alla logica del computer 
con una veloce elaborazione 
Alto Livello: Codice scritto con un linguaggio piu' vicino al modo di comunicare  
che conosciamo che necessita di un compilatore  piu' semplice per noi  


http	HyperText Transfer Protocol	Common web pages. Not encrypted
https	Secure HyperText Transfer Protocol	Secure web pages. Encrypted
ftp	     File Transfer Protocol	Downloading or uploading files
